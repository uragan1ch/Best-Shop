## Project Description

This project is a fully functional suitcase store website with responsive design,

modal windows for authorization, and a modern user interface.

## Technologies

HTML5 - semantic markup

SCSS/Sass - style preprocessor

JavaScript (ES6+) - client-side logic

ESLint - linter for code quality checking

Live Server - local development server

## Installation and Launch

# 1. Cloning the Repository

git clone https://autocode.git.epam.com/repadaniil05/fundamentals-project-template.git
cd .\fundamentals-project-template\

# 2. Installing Dependencies

npm install

# 3. Running the Project in Development Mode

npm run dev

npm run build

## Available Scripts

npm run dev | Launch in development mode with hot-reload |

npm run build | Build the project for production |

npm run build:sass | Compile SCSS to CSS |

npm run lint | Check code with ESLint and Stylelint |

## Adding New SCSS Files for Compilation

Edit the 'build:sass' script in package.json:
Example:
"build:sass": "sass src/scss/main.scss:dist/css/style.css src/scss/layouts/header.scss:dist/css/header.css src/scss/layouts/footer.scss:dist/css/footer.css"
